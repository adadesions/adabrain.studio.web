This is AdaBrain Studio. This is What we Believe.

We don't make games just to pass the time. We craft experiences that ignite imaginations, challenge perceptions, and forge unexpected connections.

จุดเริ่มต้นของ AdaBrain Studio นั้นเรามีแนวคิดที่จะสร้างเกมเพื่อการทุกคนที่ได้เล่นเกมของเรา ไม่ใช่ได้รับแค่ความสนุกเท่านั้น แต่ต้องได้สิ่งอื่นกลับไปด้วยหลังจากที่เล่นมันเสร็จ เราเลยเริ่มต้นจากแนวเกมที่เรียกกันว่า "Serious Game" เป็นเกมที่นำเสนอมุมมองอื่นของการเล่นการที่ให้มากกว่าแค่ความสนุก สิ่งนี้จึงเป็นจุดเริ่มต้นที่จริงจังของเรา จากแนวคิดที่ว่า "Textbook to Game" ทำยังไงเราจะสามารถเพิ่มพื้นที่การเรียนรู้ ได้มากกว่าแค่การอ่านหนังสือ หรือการไปเข้าห้องเรียน

เกมแรกที่เราเริ่มต้นจริงจังกันจริง ๆ คือ เกม Paideria World เป็นการที่เราพัฒนาต้นแบบกันมาหลายเวอร์ชั่นมาก ๆ เลยครับ แต่ตั้งจุดเริ่มต้นของ Pixal Art จนมาถึงทุกวันนี้ที่เกมเราเป็น 3D Environments เต็มตัว เกมนี้หากมองผิวเผิน จะรู้สึกว่าเป็นการด้านการศึกษา แต่ต้องบอกเอาไว้เลยครับ เราได้ออกแบบ วิจัยทั้งวิธีการคิด การเล่น การนำเสมอข้อมูล เพื่อที่จะทำให้เกมนี้ ก้าวข้ามผ่านการเป็น Serious Game จริง ๆ และไม่ใช่แค่ Educational Game แบบที่ในท้องตลอดมีกัน

มันเลยเป็นอะไรที่ทั้งยาก และสนุกไปพร้อมกันครับ เพราะตัวอย่างเรามีให้เราเห็นไม่ค่อยเยอะ เราเลยบุกบั่น มุ่งมั่นเพื่อให้เกม Paideria World สามารถออกสู่สายตาชาวโลก และเป็นที่ยอมรับให้ได้ครับ

ทั้งนี้ AdaBrain Studio ก็ไม่ได้หยุดอยู่แค่เกม Paideria World เท่านั้นนะ เนื่องจากพวกเราเป็นกลุ่มคนที่รักในงานเกมอย่างมาก และต้องการจะนำอุตสาหกรรมเกมไทย ไปให้ไกลให้ได้ในระดับโลก แต่จะไปยกทั้งวงการเลยก็จะเกินตัวพวกเราไปหน่อยครับ เราเลยเริ่มจากการสร้างเกมที่จะมีพื้นที่ ให้เกมไทยได้เข้าไปแข่งขันในตลาดโลกได้อย่างเกม Dilemma Deception ที่เป็นเกมแนวสยองขวัญ จิตวิทยาเชิงดาร์ก และผนวกเข้ากับวัฒนธรรมไทย ที่จะสร้างความแตกต่างให้กับผู้พบ เห็น และผู้เล่นได้อย่างแน่นนอน 

นอกจากนี้ AdaBrain Studio ยังมีความมุ่นมั่นที่จะสร้างนักพัฒนาเกมหน้าใหม่ ๆ เพื่อมาเติมเต็มให้กับอุตสาหกรรมเกมไทย ด้วยการเปิดคอร์สสอนพัฒนาเกมในทุก ๆ ระดับ ตั้งแต่เริ่มต้น ไปจนถึงการสามารถสร้างเกมด้วยตนเองได้เลย ซึ่งทีมงานที่เป็นผู้สอนก็ได้รับการ Certified จากทาง Unity Game Engine ด้วยครับ

สุดท้ายนี้ ความมุ่งมั่น และความฝันของพวกเรา คือ การได้เห็น Game Studio สัญชาติไทย ไปได้ไกลถึงในระดับโลก และเป็นการไปแบบจับมือไปด้วยกัน ไม่ใช่แค่ใคร คนใดคนหนึ่ง ดังนั้นแล้ว AdaBrain Studio จึงของเป็นอีกหนึ่งแรกผลักดัน ที่ทั้งดันตัวเอง และดันคนอื่น ไปให้ถึงฝันครับ
